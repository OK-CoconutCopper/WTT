Settings

Delete column?

Permanently removes and all its entries. This cannot be undone.

Delete column group?

The group will be removed. Its columns stay ungrouped.

Delete field?

The field and all its data across every entry will be permanently removed. This cannot be undone.

Delete entry?

This entry will be permanently removed. This cannot be undone.

Before you go…

Would you like to save a backup file first? Browser cache can be cleared unexpectedly.

Clear all entries?

Every entry in every column is removed. Structure, fields, groups, and status lists are kept. This cannot be undone.

Clear all columns?

Every column and all their entries will be permanently deleted. Column groups and status lists are kept. This cannot be undone.
